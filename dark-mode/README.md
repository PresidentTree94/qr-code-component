# Instructions for Dark Mode
These are the instructions for how to use Dark Mode in a project. By default, it is a round button with a sun in the middle. When selected, it changes to a moon, adds the `darkMode` class to the body, and saves the setting to the browser's local storage so it persists.

## Requirements
- The "dark-mode" folder and all its contents:
  - toggle-style.css
  - toggle.js

## Steps
1. Add a `<link>` to the HTML to the path of `toggle-style.css`.
2. Add a `<script>` to the HTML to the path of `toggle.js`.
3. Add a `.darkMode` to the CSS and redefine the CSS color variables. 
4. Assign colors to `#toggle` and the `svg`.
5. Add this to the HTML:
